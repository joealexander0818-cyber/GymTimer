# GymTimer - 健身训练计时器

## 项目概述
Web App 健身训练计时器，用于组间休息计时、组数记录和重量追踪。支持添加到手机主屏幕，接近原生 App 体验。

## 技术栈
- 前端：HTML + CSS + JavaScript
- 数据存储：localStorage
- 部署：GitHub Pages / Vercel

## 目录结构
